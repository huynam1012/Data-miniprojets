The win-o-meter API

Welcome to the win-o-meter API.

Endpoints

/predict, method POST

This endpoint allows you to predict the quality of the wine with a score from 0 to 10 based on physicochemical samples.

How to request this endpoint?

The expected input is a JSON with an input key followed by an array of inputs.


{input : [x1,x2,x3,x4,x5,x6,x7,x8,x9,x10,x11]} is an array of samples in the following order: 
            
x1  : fixed acidity
            
x2  : volatile acidity
            
x3  : citric acid
            
x4  : residual sugar
            
x5  : chlorides
            
x6  : free sulfur dioxide
            
x7  : total sulfur dioxide
            
x8  : density
            
x9  : pH
            
x10 : sulfates
            
x11 : alcohol

Example

JSON input example: 

          {
            "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]
          }

Result

It returns a output like so: 

          
            {
              "prediction": [y1]
            }

Request example using curl: 

          
            $ curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8], [7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' https://nam-app-wine.herokuapp.com/

Expected output: 

          
            {
              "prediction": [6, 6]
            }
